# 🧠 Autonomous AI Research Assistant

An AI-powered system that autonomously performs research, retrieves knowledge using RAG (FAISS), analyzes insights, and generates structured reports using a Multi-Agent architecture.

---

## 🚀 Features

- 🔍 Retrieval-Augmented Generation (RAG)
- 🤖 Multi-Agent System (Planner, Research, Analysis, Reasoning, Report)
- 🧠 Memory Module for storing past research
- 📊 Streamlit Dashboard for interaction
- 📄 Automated Research Report Generation

---

## 🏗️ Architecture

User Query → Planner Agent → RAG (FAISS) → Research Agent → Analysis Agent → Reasoning Agent → Report Generator → Memory → Dashboard
